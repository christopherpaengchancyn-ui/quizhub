About the Application

QuizHub is an educational quiz application designed for Bachelor of Science in Information Technology students specializing in Web and Mobile Application Development. This interactive platform helps students test and improve their knowledge in various IT-related subjects through engaging quiz challenges.

Key Features
  • Multiple Game Modes — Classic, Survival, and Speed Run modes
  • 8 IT Categories — HTML/CSS, JavaScript, Database, Mobile Dev, and more
  • Multiplayer Mode — Compete with friends in real-time
  • Host Custom Games — Create custom quizzes for others to play
  • Leaderboards — Track scores by category and game mode
  • Custom Avatars — Personalize your profile
